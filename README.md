## **ğŸš€ Getting Started**

### **Prerequisites**
- Node.js (v16 or later)
- npm or yarn

### **Installation**

1. Clone the repository
   ```bash
   git clone https://github.com/kushagra-aa/DS-Web-Portfolio.git
   ```

2. Navigate to project directory
   ```bash
   cd DS-Web-Portfolio
   ```

3. Install dependencies
   ```bash
   npm install
   ```

4. Start development server
   ```bash
   npm run dev
   ```

5. Open in browser
   ```
   http://localhost:5173
   ```
